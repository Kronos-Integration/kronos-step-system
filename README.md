# kronos-step-system

Step to execute system commands

# sample

compress streams with gzip executable

```json
  "myStep" {
    "type": "kronos-system",
    "command": "gzip",
    "args": ["-c", "-9"],
    "endpoints" : {
      "stdin" : "otherStep/out",
      "stdout" : "yetAnotherStep/in"
    }
  }
```
